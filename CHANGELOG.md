# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-26

### Added
- Initial release of simple-tiptap-texteditor
- TipTap text editor Vue 3 component with preconfigured defaults
- Fullscreen mode support
- Source code editor mode
- Customizable toolbar with extensive button options
- Support for rich text formatting (bold, italic, underline, strike, etc.)
- Support for lists (bullet and ordered)
- Support for tables with bubble menu controls
- Support for images with upload handler
- Support for YouTube video embeds
- Support for links with editing capabilities
- Typography transformations (uppercase, lowercase, capitalize)
- Text color and background color pickers
- Font family and font size selectors
- Line height control
- Text alignment options
- Horizontal rules
- Code blocks and inline code
- Subscript and superscript
- Comprehensive documentation in README.md
