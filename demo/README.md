# Demo

This folder contains demo files to showcase the Simple TipTap Text Editor in action.

## Running the Demo

```bash
# Install dependencies (from package root)
pnpm install

# Start dev server
pnpm dev
```

The demo will automatically open in your browser at `http://localhost:5173/demo/index.html`.

## Demo Files

- **App.vue** - Vue 3 component demo showcasing various editor features
- **main.ts** - Vue app entry point
- **index.html** - HTML entry file

## What's Included

The demo shows:

1. **Basic Editor** - Simple rich text editor with all features enabled
2. **Feature Grid** - Visual showcase of all built-in features
3. **Live HTML Output** - Real-time HTML preview of editor content

## Features Demonstrated

- Rich text formatting (bold, italic, underline, strikethrough, etc.)
- Image uploads with drag-and-drop
- YouTube video embedding
- Table creation and editing
- Font customization (family, size, line height)
- Text color and background color
- Typography transformations (uppercase, lowercase, camelCase, etc.)
- Source code editing
- Fullscreen mode
- Real-time HTML output
